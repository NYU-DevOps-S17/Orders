Order Demo REST API Service

GET /orders to see all the orders.

GET /orders?customer_name=Bob to see all of the orders with that customer_name.

GET /orders/{id} retrieves a order from the DB using an ID.

PUT /orders/{id} updates a order using an ID.

DELETE /orders/{id} removes a order from the DB using an ID.

POST /orders creates an new order.

POST /orders/{id} duplicates an existing order.

Example Data:

    {
        "customer_name": "Bob",
        "amount_paid": 200
    }
